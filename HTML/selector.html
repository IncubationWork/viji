CSS SELECTORS

Selectors is used to find the element you want to style.
we can divide five categories.

Simple Selectors

simple selector is select elements name, id, class

Combinator selectors

select an element based on specific relationship between them.

A CSS selector can contain more than one simple selector.
Between the simple selectors, we can include a combinator. There are four different combinators in CSS:

  • descendant selector (space)

  • child selector (>)

  • adjacent sibling selector (+)

  • general sibling selector (~)

  1. Decendant Selector (space)

    2. select and style every paragraph element that is inside div element
    (div p)

  2. Child Selector (>)

    3. select and style every paragraph element where the parent is a div element
    (div>p)

  3. Adjacent Sibling Selector (+)

    4. select and style the first paragraph element that are placed immediately after div element
    (div+p)

  4. General Sibling selector (~)

    5. select and style the all paragraph element that are next sibling of div element (div~p)

CSS Pseudo-classes

A pseudo-class is used to define a special state of an element.

For example, it can be used to:

Style an element when a user mouses over it

visited and unvisited links differently

Style an element when it gets focus

CSS Pseudo-elements

A CSS pseudo-element is used to style specified parts of an element.

For example, it can be used to:
    Style the first letter, or line, of an element
    Insert content before, or after, the content of an element

CSS Attribute Selectors

Style HTML Elements With Specific Attributes

It is possible to style HTML elements that have specific attributes or attribute values.
The attribute selector is used to select elements with a specified attribute.
